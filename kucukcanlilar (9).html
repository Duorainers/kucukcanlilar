🥧 Raspberry Pi 4 — Halka Açık Hosting Rehberi
🌍 İnternet'e Açık Site — Adım Adım
ℹ️ Pi'ye Raspberry Pi OS Lite kurulu olduğunu varsayıyoruz.
1. Nginx kur (web sunucusu):
sudo apt update && sudo apt install nginx -y
2. Site dosyasını kopyala:
sudo cp kucukcanlilar.html /var/www/html/index.html
3. Nginx başlat ve otomatik açılışı etkinleştir:
sudo systemctl enable nginx && sudo systemctl start nginx
4. Pi'nin yerel IP'sini öğren:
hostname -I
5. Ücretsiz Domain Al (DuckDNS):
→ duckdns.org'a üye ol → bir subdomain seç (ör: kucukcanlilar.duckdns.org) → Pi'nin güncel IP'sini kaydet
curl "https://www.duckdns.org/update?domains=SENINDOMAIN&token=TOKEN&ip="
6. Router Port Yönlendirme (Port Forward):
→ Router admin paneline gir (genellikle 192.168.1.1) → Port Forward → 80 (HTTP) ve 443 (HTTPS) portlarını Pi'nin yerel IP'sine yönlendir
7. SSL (HTTPS — Ücretsiz Let's Encrypt):
sudo apt install certbot python3-certbot-nginx -y && sudo certbot --nginx -d senindomain.duckdns.org
8. DDoS & Güvenlik (Cloudflare — ÜCRETSİZ):
→ cloudflare.com'a üye ol → DNS'ni Cloudflare üzerinden yönet → Otomatik DDoS engelleme aktif olur → "Under Attack Mode" açılabilir
9. Nginx Hız Limiti (Rate Limiting):
# /etc/nginx/nginx.conf içine http bloğuna ekle:
limit_req_zone $binary_remote_addr zone=site:10m rate=30r/m;
# server bloğuna ekle:
limit_req zone=site burst=20 nodelay;
10. IP Engelleme:
sudo ufw enable && sudo ufw allow 80 && sudo ufw allow 443 && sudo ufw allow 22
🔒 Kod Hırsızlığına Karşı:
→ Web sitesi kodu (HTML) prensipte kopyalanabilir, ancak tüm veriler (ürünler, siparişler, resimler) tarayıcı localStorage'da saklandığı için başkası kopyalayamaz.
→ Daha fazla koruma için JavaScript obfuscation araçları kullanabilirsin: obfuscator.io
